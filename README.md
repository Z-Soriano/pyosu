# pyosu
Parsing and Tokenizing osu! maps into working and executable python files.

# py_to_osu_helper.py
Python script that encodes a program into hitObjects and writes it in a generated_hitobjects1.txt file


# osu.tx
Defines osu grammar


# osu_runner.py
Reads, translates, and interprets hitobjects in workingEditor.osu. Writes translated program into program.osuDsl


# How to use
-Run py_to_osu_helper.py
-Replace all the hitobjects with the generated_hitobjects1 content in workingEditor.osu
-Run osu_runner.py
